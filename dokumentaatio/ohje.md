# Sovelluksen release version suorittaminen

# Asennus kääntämällä lähdekoodista

1. Asenna Nim Choosenim -työkalun avulla (lisää tarkemmat ohjeet)
2. Kloonaa repositorio (lisää tähän joku release branch)
3. Suorita repositorion kansiossa komento `nimble install` asentaaksesi sovelluksen käyttäjän kansioon tai `nimble build -d:release` luodaksesi suoritettavan binäärin repositorioon `bin` kansion alle
