This branch documents the benchmarks done to motivate making an interpreter for SDF scenes (at the start of section 4.3).

Instructions to run the benchmarks:
1. Clone the repository using `git clone -b sphere-tracing-hardcoded-scene-reload-time https://github.com/Cloudperry/nim-opengl-csg`
2. Compile the program using instructions in the main branch README.md
3. Run the command `./bin/sdf-renderer -s=<slang path>`. Replace \<slang path\> with the path where you extracted Slang.
